Firebase einrichten

Einmalige Einrichtung — danach läuft die App für alle synchron.

Schritt 1 – Firebase-Projekt erstellen

Geh auf console.firebase.google.com → "Projekt erstellen" → Name z.B. kegelclub → Google Analytics kann aus bleiben.

Schritt 2 – Firestore aktivieren

Im linken Menü: Build → Firestore Database → Create database → "Start in test mode" wählen → Region europe-west3 (Frankfurt).

Schritt 3 – Web-App registrieren

Projektübersicht → Web-Symbol </> → App-Namen eingeben → "App registrieren". Du bekommst ein firebaseConfig-Objekt mit mehreren Werten.

Schritt 4 – Werte hier eintragen

Trag die Werte aus deinem firebaseConfig unten ein:
